Data Structures in JavaScript Data structures help in organizing and storing data efficiently. Here are the main types of data structures in JavaScript: 1. Primitive Data Structures These are built-in types in JavaScript and store single values. Number → let num = 10; String → let str = "Hello"; Boolean → let isTrue = true; Null → let empty = null; Undefined → let notDefined; Symbol → let sym = Symbol("id"); BigInt → let bigNum = 12345678901234567890n; 2. Non-Primitive Data Structures These can store multiple values and are more complex. A. Linear Data Structures 1. Array → Stores multiple values in a single variable. Access elements: arr[0] Add elements: arr.push(6) Remove elements: arr.pop() Iterate: arr.forEach(item => console.log(item)) 2. Linked List → A collection of nodes, where each node contains data and a reference to the next node. Singly Linked List Doubly Linked List 3. Stack (LIFO - Last In, First Out) → Works like a pile of plates. Usage: Undo operations, function call stack. 4. Queue (FIFO - First In, First Out) → Works like a line at a grocery store. Usage: Print jobs, scheduling. 5. Deque (Double-ended Queue) → Can add/remove from both front and back. B. Non-Linear Data Structures 1. Hash Table (Object/Map) → Stores key-value pairs. 2.Set → Stores unique values. 3. Tree (Binary Tree, BST, Trie, etc.) → Hierarchical structure. Usage: Search operations, auto-suggestions. 4. Graph (Directed, Undirected, Weighted, etc.) → Represents relationships. Usage: Social networks, shortest path algorithms.
